Keyboard
  - up, down, left, right arrows to navigate
  - 1..9 keys to enter number in selected cell
  - h to show hint, a to apply hint
  - auto legals: display game tracked legals
  - manual legals: manually input legals
     - in this mode shift+1..9 will add/remove legals
  - r to start new game on game over  
Mouse
  - click to select cell
  - when game is over click to reset
